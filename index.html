About this page

This page provides the performance interface for the spatialised synthesis component of Clara Iannotta’s Troglodyte Angels Clank By.

How to use
  1. Select which instrumental part you are performing from the Instrument menu.
  2. Make sure your device has an appropriate volume setting.
  3. If starting from the beginning of the piece, press Play Next to trigger your first cue.
  4. In rehearsal you can use the plus and minus buttons to adjust which cue you want to trigger next.
Troubleshooting
For the best performance, please use Chrome for Android.
